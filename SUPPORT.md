# Support

Use the channel that matches the problem:

- Bug: open a bug report issue with reproduction details
- Feature request: open a feature request issue
- Security issue: use [SECURITY.md](SECURITY.md), not a public issue
- Usage question: start with [README.md](README.md), [docs/production-recipes.md](docs/production-recipes.md), and [docs/spring-boot-starter.md](docs/spring-boot-starter.md)

When asking for help, include:

- CacheDB version or commit
- Redis and PostgreSQL versions
- whether you are using Spring Boot starter or plain Java bootstrap
- workload shape or benchmark name
- logs, stack traces, and any relevant evidence report paths
